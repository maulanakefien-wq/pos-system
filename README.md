# نظام نقاط البيع - POS System

## وصف المشروع
نظام نقاط البيع (POS) متكامل يدعم اللغة العربية بالكامل، مع إمكانيات متقدمة للمتاجر والشركات.

## المميزات الرئيسية
- واجهة ويب سهلة الاستخدام باللغة العربية
- نظام تسجيل دخول آمن
- دعم المستخدمين المتعددين
- إدارة المبيعات والعملاء
- لوحة تحكم إحصائية
- نسخ احتياطي واستعادة البيانات

## بيانات الدخول التجريبية

### مسؤول النظام (Admin)
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

### بائع (Seller)
- اسم المستخدم: `seller`
- كلمة المرور: `123456`

## التثبيت والتشغيل

### متطلبات التشغيل
- Python 3.7+
- pip (مدير الحزم)

### خطوات التشغيل المحلي

```bash
# نسخ المشروع
git clone https://github.com/maulanakefien-wq/pos-system.git
cd pos-system

# تثبيت المتطلبات
pip install -r requirements.txt

# تشغيل التطبيق
python main.py
```

ثم افتح المتصفح وانتقل إلى: `http://localhost:5000`
